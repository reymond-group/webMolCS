Input and job submission

1) webMolCS takes the SMILES of molecules as input.

2) You can input upto 5000 molecules.

3) For each molecule in addition to the SMILES string "Name" must be provided.

4) The name of the molecule should be free of spaces or any additional special characters.

5) The SMILES string of a molecule and its name must be seperated by space or tab.

6) The valid input looks like this:

C1CCCCCC1 testMol1
CCCCCC testMol2

7) The first molecules in your list will be used as reference for similarity calculations (see section C).

8) Optionally in your input you can provide any numerical values (one for each molecule), which you may want to use for color coding of maps (see section C). These numercial values must represent third column in your input. The "Numerical value" must be seperated from "Name" by space or tag.

9) The valid input looks like this:

C1CCCCCC1 testMol1 0.02
CCCCCC testMol2 0.4

1) Copy-paste: You can simply copy-paste your input into the provided text box. Each line in the text-box must corresponds to one molecule.

2) Browse file: You can select the input file of your molecules using Choose file option. Each line in your input file must corresponds to one molecule.

In webMolCS maps are color coded according to three different criterias:

FP-Rank: First molecule in the input molecule list will be considered as query. Later, the molecules in the input list will be sorted according to their similarity (calculated using fingerprint selected by user) to the query molecule. The rank of molecules in this similarity sorted list will be use for color coding.

List-Rank: The rank of molecules calculated based on their order in the list provided by user will be use for color coding.

ActualVals: Herein the actual values provided by user will be use for color-coding.

Make sure that your input is sorted according to the criteria you are interested in.

After providing the input molecule list, click on the button corresponding to a "Fingerprint" and a "Method" of your choice.

3D-map and Visualization Functions

1) Map colors: colors are based on rank of compounds in sorted database and actual values provided by user in input.

2) FP-Rank: The first molecule in the list will be considered as reference molecule for similarity calculations.

3) List-Rank: Compounds were ranked based on their appearence in the list provided by the user.

4) ActualVals: The values provided as third column in the input list will be used for color-coding.

5) PCs: % variance covered by principal component 1, 2 and 3 (PC1-3).

6) Axes: show or hide axes.

7) MST: show or hide minmum spanning tree.

8) Point size: increase or decrease size of points.

9) Set as pivot point: rotation and camera lookup point set to the selected (locked) point in 3D-map.

10) Reset view: restore the map to starting position and reset the point size.

11) No. of cpds: set the number of compounds for automatic selection.

12) Automatic cpd list: automatically select the compounds from database.

13) ShowList: shows the list of compounds selected by user.

14) Add the compound to selection list: first lock the point by mouse double click, then click on molecule ID to put it in the selection list.

15) ExportCoord: export the coordinates, SMILES string and IDs for all the compounds.

16) Where is the reference or query molecule?: The point corresponding to the query molecule is blinking (white to red) constantly in 3D-map.



Select or lock point: mouse double click.

Unselect or unlock point: mouse double click.

Zoom: mouse middle scroll.

Rotation: press mouse left button and dragg.

Lock mouse over: double click any where.
